# I18n 类型

## I18nEntry

国际化条目。

```kotlin
@Serializable
data class I18nEntry(
    val key: String,
    val localizations: List<LocalizedString>
)
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `key` | `String` | 条目 ID |
| `localizations` | `List<LocalizedString>` | 多语言翻译列表 |

---

## LocalizedString

本地化字符串。

```kotlin
@Serializable
data class LocalizedString(
    val languageCode: Locale,
    val text: String
)
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `languageCode` | `Locale` | 语言代码 |
| `text` | `String` | 翻译文本 |

---

## TranslationStatus

翻译状态枚举。

```kotlin
enum class TranslationStatus {
    IDLE, TRANSLATING
}
```

| 值 | 说明 |
|----|------|
| `IDLE` | 空闲 |
| `TRANSLATING` | 翻译中 |
